**Supported OS:**<br>

**Run:** <br>

For Ubuntu 16.04:<br>
```
sudo apt-get update
sudo apt-get -y upgrade
sudo apt-get install -y python3-pip
git clone https://github.com/MRTech-Ev/SmS-Dynamite.git
cd SmS-Dynamite
pip3 install -r requirements.txt
python3 smsdynamite.py
```

For Turmux:
